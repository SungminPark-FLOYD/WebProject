입력후 전송 하면 ~~.jsp로 전송한다.
    전송하기 위해서 input 필드에 반드시 name속성이 있어야 한다.
    title : 마우스를 요소 위에 올려 놓으면 박스 나타남
    required : 필수항목
    autofocus : 첫 실행시 자동 커서
    pattern : 정규식으로 형식제한
    placeholder : input field에 나타나는 예시, 제목, 내용
    readonly : 값을 변경할 수 없는 field, 반드시 value속성 설정
